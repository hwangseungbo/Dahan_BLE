# Dahan_BLE
BLE를 활용한 안드로이드 어플리케이션.

<br />

### 설명
-----
유량센서의 유량 정보를 받아와 화면에 표시해주며, 펌프 제어 및 배관 세척을 통한 유량계관리를 위한 어플리케이션.
